NutriFast C964 Task 2 Overleaf Shell
====================================

Files
-----
main.tex
    Rubric-aligned report shell for the Task 2 capstone submission.

references.bib
    Starter APA bibliography file. Verify and complete all references before submission.

figures/README.txt
    Placeholder folder for screenshots, diagrams, and visual evidence.

Recommended writing order
-------------------------
1. Document the dataset first:
   raw or available nutrition data, cleaning steps, feature preparation, and the
   prepared food feature file used by the backend.

2. Document the recommendation method:
   inputs, nutrition features, ranking or scoring logic, outputs, constraints,
   evaluation evidence, and limitations.

3. Document the descriptive dashboard:
   progress metrics, analytics summaries, and at least three visualization types.

4. Add testing evidence:
   backend endpoint checks, frontend interaction screenshots, revisions, and
   accuracy or correctness checks.

5. Finish proposal and executive writing:
   letter, project proposal, executive summary, stakeholder impact, ethics,
   costs, timeline, and conclusion.

Overleaf workflow
-----------------
1. Upload the zip package or these files into a new Overleaf project.
2. Set main.tex as the main document if Overleaf does not choose it automatically.
3. Compile with pdfLaTeX and Biber support.
4. Add screenshots and diagrams under the figures folder.
5. Replace every gray placeholder note before submission.

Submission caution
------------------
The Task 2 report should describe the product that is actually implemented and
demonstrated. Keep the final paper aligned with the source code, screenshots,
dataset artifacts, testing evidence, and instructor-approved Task 1 scope.
